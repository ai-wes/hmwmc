Key model ideas

The architecture is deliberately split between a sequence substrate and a regulatory layer.

Stateful sparse substrate

Each block keeps recurrent hidden state plus a slot memory. The chunked scan reduces Python overhead while preserving recurrence semantics inside each chunk.

Episodic memory is separate from working state

The model uses both layer-local memory slots and a cross-layer episodic bank. That separation is what makes the handoff-overwrite diagnosis meaningful.

Homeostasis acts on the model, not the dataset

PNN state, circadian gating, RD stress, immune memory, and the causal tapestry all live at the controller plane. They modulate layer behavior after optimization rather than becoming part of the ordinary forward graph.

Current failure interpretation

The latest diagnostics imply the memory path is active but still brittle under overwrite and contradiction.

Handoff overwrite cliff

Recall drops as handoff count rises. That points to a write-retention problem more than a total inability to read episodic traces.

Belief revision bug

The false cue sits more recently in the numeric stream, so recurrent state leans toward recency instead of corrected truth. That is why contradiction-aware stress routing matters.

Controller is alive, but under-specified

Unlocks and PNN state cycling occur, but not yet in a way that is tightly aligned to correction events. The recent contradiction pulse patch is meant to close that gap.
