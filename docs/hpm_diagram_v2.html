v2 Architecture Changes Summary

  • NEW EntityTable — GRU-based persistent entity-state memory (B, n_entities, d_entity), soft-attention routing, separates slowly-varying facts from surprise-driven events
  • NEW Event Tape — append-only boundary snapshot buffer (max 32 entries), triggered by high surprise, stores (t, h_t, entity_state_t) for structured temporal retrieval
  • NEW Top-k routing — replaces sigmoid gate with routing logits: content_key + β·|z| + η·sticky − γ·load − δ·age, softmax → top-k mask (k=2)
  • NEW Factorized surprise — 3 independent prediction heads (ownership/rule/event), each producing its own z-score, giving the gate semantic type information
  • NEW Continuous plasticity — sigmoid(scale · age · content_mismatch · |z| − bias) replaces the OPEN/CLOSING/LOCKED state machine
  • NEW Per-state running stats — split μ/σ EMAs by slot state (OPEN/CLOSING/LOCKED), different error distributions per plasticity regime
  • NEW Slot diversity regularizer — λ · mean_pairwise_cosine(slot_keys) prevents key collapse under top-k routing
  • NEW Two-stage iterative retrieval — query embeds → family router (entity/tape/HPM) → family-specific cross-attention
  • NEW Entity History Bank — K uniformly-spaced entity-state snapshots with time embeddings for scene reconstruction
  • CHANGED Hard σ floor = 0.5 — prevents the 0.94→0.31 convergence-collapse that killed AB1 ep3
  • CHANGED Learnable gains — softplus(gain_logits) initialized to original O=1.0/C=0.5/L=0.1 values, optimizer can tune
  • CHANGED Content-conditional writes — slot_key + content_query with softmax(q·k/√d) write weighting, load-balance penalty
  • CHANGED Gate MLP input — now takes [h_t ; z_own ; z_rule ; z_event ; state_embed] instead of scalar z
  • CHANGED Retroactive binding — write = learned mixture over causal window h_{t-W:t} instead of h_t alone
  • REMOVED Hard force-unlock — replaced by soft age penalty in routing logits (no cliff behavior)
  • REMOVED State machine (optional) — continuous plasticity subsumes O/C/L with a smooth scalar, still available as fallback
